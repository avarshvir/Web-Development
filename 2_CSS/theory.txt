CSS -> Castcading Style Sheet. It is used for styling, beautify the website.

Types:
- Inline
- Internal
- External

-----------------------------------------------------------------------------------
- Inline:
syntax:
<h1 style = "property:value">content</h1>
<p style = "propert:value; property:value;">content</p>
example:
<h1 style = "color:red">Hi</h1>
<p style = "color:red; background-color:yellow;">How are you</p>

-----------------------------------------------------------------------------------
- Internal:
syntax:
<head>
    <style>
        h1{
            property:value;
        }
    </style>
<head>

-----------------------------------------------------------------------------------
- External:
syntax:
file_name- style.css
h1{
    property:value;
}
